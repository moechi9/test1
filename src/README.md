# お問い合わせフォーム

## 環境構築

### Dockerビルド

- docker-compose up -d --build

### Laravel環境構築

- docker-compose exec php bash
- composer install
- .env.exampleファイルから.envを作成し、環境変数を変更
- php artisan key generate
- php artisan migrate
- php artisan db:seed
